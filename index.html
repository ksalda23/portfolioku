Profil

Saya Kartiko Syla Alda, saya lahir di Demak tahun 2002. Saya tinggal di Mranggen bersama kedua orang tua saya. Saya pernah bersekolah di SMK Bhakti Nusantara. Disana saya mengambil kejuruan Multimedia (Teknik Informatika).

Selama saya bersekolah di SMK Bhakti Nusantara, saya pernah magang selama 3 bulan di Kantor Gubernur Jawa Tengah. Di Kantor Gubernur Jawa Tengah, saya ditempatkan di Asisten Tata Usaha Wakil Gubernur Jawa Tengah. Selama magang, Saya ditugaskan untuk pengelolaan dan pengarsipan surat-surat masuk dan keluar.
